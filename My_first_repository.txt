1. Що таке Git?
Git – це програмне забезпечення для контролю версій написана мовою С. Git дозволяє вам відстежувати кожну зміну, яку ви робите під час розробки програмного забезпечення.

2. В якому статусі можуть перебувати файли при роботі з Git?
Файли можуть перебувати в стані: збережений у коміті(commited), змінений(modified) та індексований (staged).

3. Для чого використовується команда add?
За допомогаю цієї команди можна додати необхідний файл до репозиторію.

4. Для чого використовується команда status?
Ця команда перевіряє, чи додані файли(зелений колір - додані, червоний - не додані).

5. Для чого використовується команда commit?
Фіксація стану сховища в певний момент часу.

6. Для чого використовується команда init?
Це ініціалізація Git у папці проекту.

7. Що необхідно писати в коментарях комітів?
Треба писати файли які нам необхідно зберегти.

8. Як переглянути список комітів? 
Використовується команда git log

9. Для чого використовується команда checkout?
Команда git checkout використовується для переключення гілок та отримання вмісту до вашої робочої директорії.
